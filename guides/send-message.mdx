## Send a text message

Send to a known WhatsApp JID:

<CodeGroup dropdown>

```bash cURL
curl -X POST \
  -H "Authorization: Bearer $WHATSAPP_USE_API_KEY" \
  -H "Content-Type: application/json" \
  "$WHATSAPP_USE_BASE_URL/v1/whatsapp/messages/send" \
  -d '{
    "contact_id": "14155550123@s.whatsapp.net",
    "text": "Hello from WhatsApp Use"
  }'
```

```javascript JavaScript
const res = await fetch(`${process.env.WHATSAPP_USE_BASE_URL}/v1/whatsapp/messages/send`, {
  method: "POST",
  headers: {
    Authorization: `Bearer ${process.env.WHATSAPP_USE_API_KEY}`,
    "Content-Type": "application/json",
  },
  body: JSON.stringify({
    contact_id: "14155550123@s.whatsapp.net",
    text: "Hello from WhatsApp Use",
  }),
});

console.log(await res.json());
```

```python Python
import os
import requests

res = requests.post(
    f"{os.environ['WHATSAPP_USE_BASE_URL']}/v1/whatsapp/messages/send",
    headers={"Authorization": f"Bearer {os.environ['WHATSAPP_USE_API_KEY']}"},
    json={
        "contact_id": "14155550123@s.whatsapp.net",
        "text": "Hello from WhatsApp Use",
    },
)
res.raise_for_status()
print(res.json())
```

```go Go
package main

import (
	"bytes"
	"fmt"
	"io"
	"net/http"
	"os"
)

func main() {
	body := []byte(`{"contact_id":"14155550123@s.whatsapp.net","text":"Hello from WhatsApp Use"}`)
	req, _ := http.NewRequest("POST", os.Getenv("WHATSAPP_USE_BASE_URL")+"/v1/whatsapp/messages/send", bytes.NewReader(body))
	req.Header.Set("Authorization", "Bearer "+os.Getenv("WHATSAPP_USE_API_KEY"))
	req.Header.Set("Content-Type", "application/json")

	resp, err := http.DefaultClient.Do(req)
	if err != nil {
		panic(err)
	}
	defer resp.Body.Close()
	out, _ := io.ReadAll(resp.Body)
	fmt.Println(string(out))
}
```

Reply in an existing conversation:

```bash
curl -X POST \
  -H "Authorization: Bearer $WHATSAPP_USE_API_KEY" \
  -H "Content-Type: application/json" \
  "$WHATSAPP_USE_BASE_URL/v1/whatsapp/messages/send" \
  -d '{
    "conversation_id": "conv_14155550123swhatsappnet",
    "text": "Following up here"
  }'
```

## React to a message

```bash
curl -X POST \
  -H "Authorization: Bearer $WHATSAPP_USE_API_KEY" \
  -H "Content-Type: application/json" \
  "$WHATSAPP_USE_BASE_URL/v1/whatsapp/messages/react" \
  -d '{
    "conversation_id": "conv_14155550123swhatsappnet",
    "message_id": "3EB0...",
    "emoji": "\uD83D\uDC4D"
  }'
```

Use an empty `emoji` to clear the reaction.

## Create a group

```bash
curl -X POST \
  -H "Authorization: Bearer $WHATSAPP_USE_API_KEY" \
  -H "Content-Type: application/json" \
  "$WHATSAPP_USE_BASE_URL/v1/whatsapp/groups" \
  -d '{
    "name": "Launch updates",
    "participants": ["14155550123@s.whatsapp.net"]
  }'
```
